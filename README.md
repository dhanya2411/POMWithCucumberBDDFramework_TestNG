# POMWithCucumberBDDFramework_TestNG using Selenium with Java, TestNG and Maven-
This is a sample project to demonstrate the Page Object Model with Cucumber BDD framework and how it can used in selenium to automate any application.
TestNG is used as a test framework.

Dependency
Java
Maven

###libraries used
Selenium
TestNG
Gherkin
Reporting
JVM Dependencies

### Steps to clone and execute the tests
```
git clone https://github.com/dhanya2411/POMWithCucumberBDDFramework_TestNG.git
cd POMWithCucumberBDDFramework_TestNG
mvn clean test
